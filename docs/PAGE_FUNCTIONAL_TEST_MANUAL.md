# ✅ 페이지 기능 테스트 검증 가이드

**작성일**: 2025.12.21  
**목적**: 각 페이지 기능을 수동으로 검증하는 실전 가이드

---

## 🚀 빠른 시작

### 테스트 서버 준비
```bash
# 터미널 1: API 서버 시작
pnpm api:dev

# 터미널 2: 웹앱 서버 시작
pnpm web:dev

# 터미널 3: 데이터베이스 초기화 (필요시)
pnpm db:seed
```

### 브라우저 열기
```
http://localhost:4200
```
