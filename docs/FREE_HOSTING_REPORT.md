# MVP 무료 호스팅 조사 보고서

> 조사일: 2026-01-09
> 대상: 물류 ERP 시스템 MVP 배포

## 요약

본 프로젝트(Angular + NestJS + PostgreSQL + Redis)를 **완전 무료**로 호스팅할 수 있는 조합을 조사했습니다.

### 추천 조합

| 구성요소 | 추천 서비스 | 무료 티어 |
|----------|-------------|-----------|
| **프론트엔드 (Web)** | Cloudflare Pages | 무제한 요청, 500회 빌드/월 |
| **프론트엔드 (Mobile PWA)** | Cloudflare Pages | 동일 |
| **백엔드 (NestJS)** | Render | 750시간/월, 자동 슬립 |
| **데이터베이스 (PostgreSQL)** | Neon | 3GB 스토리지, 서버리스 |
| **캐시 (Redis)** | Upstash | 50만 명령/월 |

**예상 월 비용: $0**

|------|-----------|
| 요청 수 | **무제한** |
| 대역폭 | **무제한** |
| 빌드 | 500회/월 |
| 동시 빌드 | 1개 |
| 사이트 수 | 무제한 |
| 커스텀 도메인 | 무제한 |

**단점:**
- 빌드 시간 제한 (20분)
- 서버리스 함수 기능 제한적

**단점:**

## 2. 백엔드 호스팅 비교

### 2.1 Render ⭐ 추천

| 항목 | 무료 티어 |
|------|-----------|
| 실행 시간 | 750시간/월 |
| 메모리 | 512MB |
| CPU | 공유 |
| 자동 배포 | Git 연동 |
| SSL | 자동 |

**장점:**
- 신용카드 불필요
- NestJS 완벽 지원
- 자동 HTTPS
- 환경 변수 관리 편리

**단점:**
- **15분 비활성 시 슬립** (콜드 스타트 ~30초)
- 스토리지 없음

**슬립 문제 해결책:**
- 크론잡으로 주기적 핑 (UptimeRobot 무료)
- 프론트엔드에서 로딩 스피너 표시

### 2.2 Fly.io

| 항목 | 무료 티어 |
|------|-----------|
| 무료 크레딧 | $5/월 상당 |
| VM | 3개 공유 CPU |
| 메모리 | 256MB x 3 |
| 스토리지 | 3GB 볼륨 |

**장점:**
- 정적 IP 제공
- 글로벌 엣지 배포
- 슬립 없음 (24시간 가동)
- Docker 지원

**단점:**
- **신용카드 필수**
- 설정이 복잡함
- 무료 범위 초과 시 자동 과금

### 2.3 Railway

| 항목 | 무료 티어 |
|------|-----------|
| 트라이얼 | $5 크레딧 (일회성) |
| Hobby | $5/월~ |

**장점:**
- 가장 쉬운 배포 경험
- PostgreSQL/Redis 원클릭 추가
- 환경 간 연결 자동화

**단점:**
- **무료 티어 없음** (트라이얼만)
- MVP 이후 유료 전환 필요

### 2.4 Leapcell (신규)

| 항목 | 무료 티어 |
|------|-----------|
| 비용 | 무료 |
| 신용카드 | 불필요 |
| 지원 | Node.js, Python, Go |
| 부가 서비스 | PostgreSQL, Redis 포함 |

**장점:**
- 신용카드 없이 가입
- DB + Redis 번들 제공
- 신규 서비스로 관대한 정책

**단점:**
- 신규 서비스 (안정성 미검증)
- 문서/커뮤니티 부족

| 항목 | 무료 티어 |
|------|-----------|
| 스토리지 | **3GB** |

| 브랜치 | 10개 |
| 프로젝트 | 1개 |

**장점:**
- **가장 큰 무료 스토리지**

- Git 스타일 브랜칭
- 콜드 스타트 ~1초

**단점:**
- 리전 제한 (미국/유럽)
- 일부 확장 기능 제한

### 3.2 Supabase

| 항목 | 무료 티어 |
|------|-----------|
| 스토리지 | 500MB |
| API 요청 | 무제한 |
| 인증 사용자 | 50,000명/월 |
| 스토리지 (파일) | 1GB |

**장점:**
- 인증/스토리지/실시간 통합
- REST/GraphQL API 자동 생성
- 대시보드 우수

**단점:**
- 스토리지 500MB로 제한적

## 6. 배포 전략

### 6.1 권장 배포 순서

```
1. Neon에 PostgreSQL 생성
   └── DATABASE_URL 획득

2. Upstash에 Redis 생성
   └── REDIS_URL 획득

3. Render에 NestJS 배포
   ├── 환경 변수 설정
   └── 자동 배포 연결

4. Cloudflare Pages에 프론트엔드 배포
   ├── Web 앱 배포
   └── Mobile PWA 배포
```

### 6.2 슬립 방지 설정 (Option A)

UptimeRobot (무료)으로 5분마다 헬스체크 핑:

```
대상 URL: https://your-api.onrender.com/api/v1/health
간격: 5분
```
